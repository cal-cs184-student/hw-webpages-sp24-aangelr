Reflection

We no longer think doing a 3D rendering of the fonts is reasonable anymore given the time we have and the amount of work it would take. Our goal moving forward is to build a rasterizer that is able to draw our characters to the frame buffer using the FreeType interface for the character information. We also want to do the rendering through the rasterizer since we are currently using matplotlib to verify that we are correctly parsing the control points.
