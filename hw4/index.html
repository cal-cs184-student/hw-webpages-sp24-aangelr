Overview

Give a high-level overview of what you implemented in this project. Think about what you've built as a whole. Share your thoughts on what interesting things you've learned from completing the project.

In this project, we constructed a realistic cloth simulation using the point mass and spring structure. All of our point masses were connected to neighboring point masses with springs of the correct constraint types. We applied external forces to simulate gravity and pinned point masses. For all objects in the scene, we also checked for collision with that object and every point mass and correctly placed the point mass on the outside of the object without letting it clip through. In order to handle self-collision, we implemented a spatial map so that we apply corrective forces to a point mass relative to other point masses within the same spatial region. This ensured that point masses would not clip past each other.
